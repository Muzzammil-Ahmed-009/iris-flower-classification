# Iris Flower Classification

Multi-class classification model to predict Iris flower species using Scikit-learn.

##  Project Overview
- **Dataset:** Iris Dataset (150 samples, 4 features, 3 classes)
- **Goal:** Classify flowers as Setosa, Versicolor, or Virginica
- **Models Used:** Logistic Regression, K-Nearest Neighbors (KNN)
- **Best Accuracy:** 96.67%

## Tech Stack
